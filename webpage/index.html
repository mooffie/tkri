Tkri

tkri is a GUI front-end to Ruby's RI, and to FastRI. By default it runs qri, the FastRI command, and shows its output inside a window where every word is a hyperlink.

The purpose of tkri is to minimize the keystrokes you make when navigating in rdoc documentation.

Installation

Tkri is shipped as a gem. At your system prompt type:

gem install tkri

Then launch the program by typing tkri at the system prompt. Check out this program's Help menu for further details.

Requirements

This program uses the Tk user interface toolkit.
